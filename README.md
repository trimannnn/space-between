# The Space Between

A 3D spatial mind palace for thoughts, fragments, and creative process.

**Live:** https://trimannnn.github.io/space-between

---

## What it is

An immersive 3D sphere where each node is a room — a container for images, text fragments, and thoughts. Orbit the sphere, click a node to open it, add images and writing. The space remembers everything.

## Features

- **3D mind palace** — nodes connected by opacity-tiered flow lines, nearest-neighbour spatial layout
- **Rooms** — each node holds images, text fragments, and a thought that floats in 3D space
- **Three modes** — dark, light (cool grey-teal), and play (iridescent bubbles + flow particles)
- **Glowing cursor** — canvas-drawn blob that changes colour per mode
- **Auto-save** — everything persists in localStorage, survives refresh
- **Export / Import** — download your space as JSON, reload it anywhere
- **Connection weight** — nodes grow with content
- **Sphere sheen** — sparkle layer on the wireframe shell

## How to use

- **Drag** — orbit the sphere
- **Scroll** — zoom
- **Click a node** — open the room modal
- **New Room** (top right) — create a node, modal opens immediately
- **+ thought** — add text fragments inside a room, they float on the node in 3D
- **Paste image** — Cmd/Ctrl+V when modal is open
- **Theme buttons** — bottom right, moon / half-moon / 3D sphere

## Stack

Pure HTML/CSS/JS — single file, no build step, no dependencies except Three.js (bundled inline).
